FastAPI
FastAPI is a modern, high-performance Python web framework for building APIs, created by Sebastián Ramírez (Tiangolo) and first released in 2018. It is designed to be fast to code, easy to use, and production-ready out of the box.

Core Philosophy
FastAPI is built on two key ideas: speed and developer experience. It leverages Python's type hint system to provide automatic validation, serialization, and documentation — eliminating a huge amount of boilerplate code that other frameworks require.

Key Features
Performance
FastAPI is one of the fastest Python frameworks available, on par with NodeJS and Go. This is because it's built on top of Starlette (for the web layer) and Pydantic (for data validation), both of which are highly optimized. It is fully asynchronous, built natively on Python's async/await syntax via ASGI (Asynchronous Server Gateway Interface).
Automatic Data Validation
Using Python type hints and Pydantic models, FastAPI automatically validates incoming request data (query params, path params, request bodies, headers, cookies). If the data doesn't match the expected types, it returns a clear, structured error response automatically.
Automatic Interactive Documentation
FastAPI auto-generates two interactive API documentation UIs out of the box with zero extra configuration:

Swagger UI — available at /docs
ReDoc — available at /redoc

These are generated directly from your code's type hints and docstrings, and stay in sync with your API automatically.
Dependency Injection
FastAPI has a powerful, intuitive built-in dependency injection system. Dependencies (like database sessions, authentication logic, or shared utilities) can be declared as function parameters and FastAPI resolves and injects them automatically. Dependencies can be nested and reused across routes.
Security & Authentication
FastAPI has built-in support for standard security schemes including OAuth2 (with password and bearer flows), HTTP Basic Auth, API keys, and OpenID Connect. These integrate cleanly with the dependency injection system.
WebSockets
FastAPI supports WebSocket connections natively, making it suitable for real-time applications like chat systems or live dashboards.
Background Tasks
You can define background tasks that run after returning a response — useful for sending emails, processing data, or logging without blocking the response.
